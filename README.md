# 📄 Projeto Web com HTML, CSS e Bootstrap

## 📌 Sobre o Projeto

Este projeto foi desenvolvido utilizando **HTML5**, **CSS3** e **Bootstrap**, com o objetivo de criar uma interface moderna, responsiva e organizada.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5

## 📖 Descrição das Tecnologias

### HTML5

O **HTML (HyperText Markup Language)** é a linguagem responsável pela estrutura das páginas web. Com ele é possível criar títulos, textos, imagens, tabelas, formulários e diversos outros elementos.

### CSS3

O **CSS (Cascading Style Sheets)** é utilizado para estilizar a página, definindo cores, fontes, espaçamentos, animações e o layout dos elementos.

### Bootstrap

O **Bootstrap** é um framework front-end que facilita a criação de interfaces responsivas. Ele oferece componentes prontos, sistema de grid, botões, formulários, menus e diversas classes utilitárias que aceleram o desenvolvimento.
